## 📥 Clone the Repository

```bash
git clone https://github.com/helinatefera/VQAGen-Demo.git
cd VQAGen-Demo
````

---

## 🛠️ Setup & Run

Install all required dependencies:

```bash
pip install -r requirements.txt
```

Run the graphical interface:

```bash
python gui_live.py
```
